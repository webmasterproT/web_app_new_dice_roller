How to Use the Dice Roller

Rolling Dice

To roll dice, follow these steps:

  1. Select the number of dice you want to roll.
  2. Choose the type of dice from d2 up to d100.
  3. Add any modifiers to your roll if necessary.
  4. Click the 'Roll' button to see the result.

Adding Dice Rollers

You can add additional dice rollers by clicking the '+' button next to the dice roller. This allows you to roll multiple sets of dice at once.

Using the Results Log

The results log will display the history of your rolls with timestamps. You can save and export this log to a PDF by clicking the 'Save Log' button.

Notes Section

Keep track of game details or any notes you need in the notes section provided.
